# 🌫️ 광고형 향수 브랜드 웹앱 프로젝트

> 세상에 하나뿐인 주문제작 향수를 위한 감성 브랜딩 중심 웹앱 프로젝트

---

## 📌 프로젝트 소개

향이라는 감각적 요소를  
웹 인터랙션과 브랜드 경험으로 확장한 프로젝트입니다.

단순 제품 소개가 아닌:

- 브랜드 무드 전달
- 감성 중심 UX
- 시네마틱 인터랙션
- 브랜드 아이덴티티 구축
- 사용자 경험 중심 디자인

에 집중하여 제작했습니다.

---

## 🎨 브랜드 컨셉

> “향은 기억을 남긴다.”

사용자의 감정과 분위기를 기반으로  
세상에 존재하지 않는 단 하나의 향을 제작하는  
커스텀 핸드메이드 향수 브랜드입니다.

---

## 🧠 디자인 방향성

감성과 분위기를 시각적으로 표현하기 위해:

- 무드 중심 레이아웃
- 부드러운 모션 인터랙션
- 감성 타이포그래피
- 브랜드 컬러 시스템
- 몰입형 UI 흐름

을 중심으로 디자인했습니다.

---

## ✨ CI / BI 디자인

브랜드 로고 및 아이덴티티를 직접 제작했습니다.

### 포함 내용

- 로고 디자인
- 심볼 디자인
- 타이포그래피 시스템
- 컬러 시스템
- 브랜드 무드 방향성

---

## 🖌️ 로고 제작 과정

Figma 기반으로 로고 제작 과정을 정리했습니다.

```txt
스케치 → 도형 제작 → 그리드 정리 → 타이포 적용 → 최종 로고 완성
```
